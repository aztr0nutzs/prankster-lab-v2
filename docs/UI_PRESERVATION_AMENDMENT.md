# UI_PRESERVATION_AMENDMENT.md

Last updated: 2026-05-07

## STRICT UI PRESERVATION AMENDMENT

The Prankster Lab app must maintain a premium dark neon cyberpunk prank-lab identity.

Final QA phase rule:
Do not redesign UI, simplify UI, or remove visual systems. Only fix bugs, polish issues, accessibility, performance, validation, and documentation.

## FORBIDDEN REGRESSIONS

Do not:
- replace custom UI with plain Material pages
- flatten layered surfaces
- remove waveform systems
- remove glow systems
- remove reactor interactions
- remove Sound Forge workstation visuals
- replace custom dock navigation with default tabs
- downgrade animated interfaces into plain columns
- replace premium cards with plain lists
- reduce visual hierarchy
- remove neon styling

## REQUIRED VISUAL DNA

Preserve:
- dark cyberpunk aesthetic
- neon edge lighting
- layered surfaces
- animated transitions
- waveform headers and waveform cards
- premium bottom dock navigation
- glowing category chips
- HUD-inspired composition
- animated reactor core
- rich interaction feedback

## REACTOR CORE PROTECTION

The Reactor Core must not become:
- a plain circular button
- a static image
- a generic play control

Required systems to preserve:
- category arcs
- playback-reactive visuals
- waveform halo
- charge/cooldown/error states
- metadata overlays

## SOUND FORGE PROTECTION

Sound Forge must remain:
- visually rich
- animated
- workstation-like
- experimental
- waveform-forward

It must not become:
- a plain slider list
- a generic settings screen
- a static form layout
