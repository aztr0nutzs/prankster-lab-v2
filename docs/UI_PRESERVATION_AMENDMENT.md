# UI_PRESERVATION_AMENDMENT.md

## STRICT UI PRESERVATION AMENDMENT

The Prankster app MUST maintain a premium neon cyberpunk prank-lab visual identity.

Forbidden regressions:
- plain Material layouts
- flat black screens
- empty spacing
- generic cards
- simple button columns
- text-only layouts
- spreadsheet-style sound lists

Required visual DNA:
- neon edge lighting
- layered surface depth
- waveform systems
- animated glow systems
- cyberpunk HUD styling
- animated dock navigation
- reactor-style interactions
- premium spacing
- glowing controls
- dynamic overlays

## REACTOR CORE PROTECTION

The Reactor Core MUST NOT become:
- a plain circular button
- static image
- generic play control

Required future systems:
- category arcs
- playback-reactive visuals
- waveform halo
- haptic interaction
- cooldown/error states
- metadata overlays

## SOUND FORGE PROTECTION

Sound Forge MUST remain:
- visually rich
- animated
- workstation-like
- experimental

It MUST NOT become:
- a plain slider list
- generic settings screen
- static form layout
